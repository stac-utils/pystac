# pystac-ext-raster

[PySTAC](https://pypi.org/project/pystac/) extension package for the [Raster Extension](https://github.com/stac-extensions/raster).
This extension provides fields for describing raster band metadata, including sampling type, bits per sample, spatial resolution, scale, offset and band histogram.

## Supported versions

- [v2.0.0](https://stac-extensions.github.io/raster/v2.0.0/schema.json)
- [v1.1.0](https://stac-extensions.github.io/raster/v1.1.0/schema.json)
- [v1.0.0](https://stac-extensions.github.io/raster/v1.0.0/schema.json)

## Versioning

This package's version corresponds to the version of the extension specification it targets.
When we release updates to the package code without changing the target extension version, we use [post releases](https://packaging.python.org/en/latest/discussions/versioning/#post-releases), e.g. `1.1.0.post1`.
